Dokumentation för min API

Alla olika routes

  • GET /users - Retunerar alla avnändare som finns i databasen.

  • GET /users/:id - Retunerar en specefik användare baserat på deras id-nummber, byt ut ":id" mot ett nummer.

  • POST /users - Skapar en ny användare accepterar JSON objekt på formatet {"username": "", "password": "", "name": "", "email": ""}. username är obligatoriskt, annars skickas 422-meddelande (unprocessable entity).
